<h1>Copy File Contents</h1>

<h2>Overview</h2>
<p>Copy the entire contents of any file to your clipboard with a single right-click — no need to open the file first.</p>

<h3>Key Features</h3>
<ul>
<li>Copy full file contents without opening the file</li>
<li>Works on any file type in the Explorer</li>
<li>Restricted to single-file selection (multi-select is not supported)</li>
<li>Preserves exact file content including encoding and line endings</li>
<li>Instant clipboard write with success notification</li>
</ul>

<h2>Usage</h2>
<ol>
<li>Right-click a single file in the Explorer sidebar</li>
<li>Select <strong>Additional Context Menus ▶ Copy File Contents</strong></li>
<li>The file contents are now in your clipboard</li>
</ol>

<h2>Use Cases</h2>
<ul>
<li>Quickly share file contents without opening an editor tab</li>
<li>Copy configuration files, templates, or snippets to paste elsewhere</li>
<li>Copy file contents before destructive edits as a manual backup</li>
<li>Paste entire file contents into chat tools, documentation, or issue trackers</li>
</ul>
