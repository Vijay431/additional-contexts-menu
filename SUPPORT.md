## Before Asking for Help

To help us help you more effectively, please:

1. **Search existing resources**
   - Check if your question has already been answered in Discussions
   - Search closed issues for similar problems
   - Review the documentation

2. **Gather information**
   - VS Code version
   - Extension version
   - Operating system
   - Relevant error messages or logs
   - Steps to reproduce the issue

3. **Try basic troubleshooting**
   - Restart VS Code
   - Disable other extensions to check for conflicts
   - Check the VS Code Output panel for error messages
   - Verify your VS Code settings

## How to Ask Questions

### For General Questions (Use Discussions)

When asking a question in GitHub Discussions:

1. **Use a clear, descriptive title**
   - ✅ Good: "How to configure custom context menu items?"
   - ❌ Bad: "Help needed"

2. **Provide context**
   - What are you trying to accomplish?
   - What have you already tried?
   - What documentation have you consulted?

3. **Include relevant details**
   - Code snippets (use code blocks)
   - Configuration settings
   - Screenshots if applicable

### For Bug Reports (Use Issues)

When reporting a bug in GitHub Issues:

1. **Use the bug report template**
   - Fill out all required sections
   - Provide clear reproduction steps
   - Include expected vs. actual behavior

2. **Provide a minimal reproduction**
   - Simplify the issue to the smallest possible example
   - Remove unrelated code or configuration

3. **Include environment details**
   - VS Code version
   - Extension version
   - Operating system
   - Other relevant extensions

## Response Times

This is an open-source project maintained by volunteers. Response times may vary:

- **General questions**: We aim to respond within 5 business days
- **Bug reports**: We aim to acknowledge within 5 business days
- **Security issues**: We aim to respond within 48 hours (see [SECURITY.md](SECURITY.md))

Please be patient and respectful. Maintainers and community members contribute their time voluntarily.

## What NOT to Use Issues For

Please **do not** use GitHub Issues for:

- ❌ General questions (use Discussions instead)
- ❌ Feature requests without discussion (propose in Discussions first)
- ❌ Support requests (use Discussions)
- ❌ Security vulnerabilities (use private disclosure via [SECURITY.md](SECURITY.md))
