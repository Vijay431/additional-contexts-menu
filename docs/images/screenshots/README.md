## Recording Guidelines

- **Dimensions**: 1280x800px
- **Frame rate**: 15fps
- **Max file size**: 2MB per GIF
- **Theme**: VS Code Dark+ (default dark theme)
- **Zoom**: 125% for readability

## Tools

- **macOS**: [Kap](https://getkap.co/)
- **Windows**: [ScreenToGif](https://www.screentogif.com/)
- **Linux**: [Peek](https://github.com/phw/peek)

## Before Recording

1. Set VS Code to Dark Theme (Default Dark+)
2. Increase editor zoom to 125%
3. Hide minimap (View > Appearance > Minimap > Hide)
4. Set font size to 14px
