# Save All Files

Right-click anywhere in the editor and choose **Save All**, or press `Ctrl+Alt+Shift+A` (`Cmd+Alt+Shift+A` on Mac).

All open modified files are saved at once. Read-only files are skipped automatically.

A notification reports how many files were saved, skipped, or failed — so you always know the outcome.

**Tip:** Run this before a commit or build to make sure nothing is left unsaved.
