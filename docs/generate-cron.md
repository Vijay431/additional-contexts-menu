# Generate Cron Expression

Right-click anywhere and choose **Generate Cron Expression** to build a cron schedule interactively.

Answer the prompts (minute, hour, day, month, weekday) and the extension produces a valid cron string.

**Example output:**

```
0 9 * * 1-5
```

_Runs at 09:00, Monday through Friday._

The expression is inserted directly at your cursor position.
