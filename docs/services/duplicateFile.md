<h1>Duplicate File</h1>

<h2>Overview</h2>
<p>Duplicate any file from the Explorer with a single right-click — no need to open it or use the file system manually.</p>

<h3>Key Features</h3>
<ul>
<li>Creates a duplicate alongside the original in the same directory</li>
<li>Duplicate is named <code>&lt;basename&gt;-duplicate&lt;ext&gt;</code> (e.g. <code>index.ts</code> → <code>index-duplicate.ts</code>)</li>
<li>Auto-increments if the name already exists: <code>index-duplicate-1.ts</code>, <code>index-duplicate-2.ts</code>, …</li>
<li>Works on any file type in the Explorer</li>
<li>Restricted to single-file selection (folders and multi-select are not supported)</li>
</ul>

<h2>Usage</h2>
<ol>
<li>Right-click a single file in the Explorer sidebar</li>
<li>Select <strong>Additional Context Menus ▶ Duplicate File</strong></li>
<li>The duplicate appears in the same directory with a <code>-duplicate</code> suffix</li>
</ol>

<h2>Use Cases</h2>
<ul>
<li>Quickly create a backup copy before experimenting with changes</li>
<li>Scaffold a new file from an existing template</li>
<li>Duplicate configuration files for different environments</li>
<li>Create test fixtures based on existing files</li>
</ul>
