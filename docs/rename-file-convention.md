# Rename File to Convention

Open the Command Palette (`Ctrl+Shift+P`) and run **Additional Context Menus: Rename File to Convention**.

Choose a naming style — the current file is renamed instantly.

| Style | Example |
|---|---|
| kebab-case | `user-profile-card.ts` |
| camelCase | `userProfileCard.ts` |
| PascalCase | `UserProfileCard.ts` |

**Example:** `UserProfileCard.tsx` → `user-profile-card.tsx` (kebab-case)
