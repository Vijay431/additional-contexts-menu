# Open in Terminal

Right-click any file or folder in the editor or explorer and choose **Open in Terminal**.

The extension will open a terminal at the selected location using your configured terminal type:

- **Integrated** — opens VS Code's built-in terminal panel
- **External** — launches your system's default terminal application
- **System Default** — uses the OS default terminal

**Configure the behavior** in Settings → Additional Context Menus → Terminal.
